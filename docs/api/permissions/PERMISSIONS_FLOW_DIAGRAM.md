### UserPermissions Object (Flutter)

```dart
UserPermissions {
  roles: ["admin", "moderator"],
  permissions: [
    "delete patient",
    "edit posts",
    "view reports"
  ]
}
```

### Cached in SharedPreferences

```json
{
  "roles": ["admin"],
  "permissions": ["delete patient", "edit posts"],
  "cached_at": "2024-10-04T12:00:00Z"
}
```

### API Response Structure

```json
{
  "value": true,
  "roles": {
    "admin": true,
    "moderator": false
  },
  "permissions": {
    "delete patient": true,
    "edit posts": true,
    "view reports": false
  }
}
```
